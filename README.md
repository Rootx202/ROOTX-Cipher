# 🛡️ ROOTX Cipher - Advanced Cryptography Toolkit

![ROOTX Cipher Logo](RootXCipherlogo.png)

**ROOTX Cipher** is a professional, high-performance desktop application built with Python and PyQt6. It provides a comprehensive suite of cryptographic tools, from modern industry standards to classical educational ciphers. It supports both plain text processing and direct binary file manipulation.

---

## 🚀 Key Features

### 🔐 Modern Encryption
*   **Symmetric:** AES, DES, 3DES, Blowfish, Twofish, Camellia.
*   **Asymmetric:** RSA, ECC (Elliptic Curve), DSA, ElGamal, Diffie-Hellman Key Exchange.
*   **Stream Ciphers:** ChaCha20, Salsa20, RC4.

### 🔢 Hashing & Integrity
*   **Algorithms:** SHA-256, SHA-512, SHA-3, MD5, BLAKE2, RIPEMD-160, Whirlpool.
*   **File Hashing:** Efficient streamed processing for large files without RAM exhaustion.

### 📜 Classical & Utility
*   **Classical Ciphers:** Caesar, Vigenere, ROT13, Atbash, Affine.
*   **Encoding:** Base64, Hexadecimal, Binary, URL Encoding.

### 🎨 User Experience
*   **Professional UI:** Modern design with Dark and Light theme support.
*   **File Mode:** Process files directly by path without loading binary content into text boxes.
*   **Smart Decryption:** Intelligent workflow that prioritizes previous results for decryption.
*   **Multilingual Installer:** Ready-to-use Windows installer with Arabic and English support.

---

## 🛠️ Technology Stack

*   **Language:** Python 3.11+
*   **UI Framework:** PyQt6 (Qt v6)
*   **Crypto Engine:** `cryptography` & `hashlib`
*   **Build Tools:** PyInstaller, Inno Setup

## 📦 Installation & Setup

### For Developers
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/ROOTX-Cipher.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python main.py
   ```

### For Users
Download the `ROOTX_Cipher_Setup.exe` from the [Releases](https://github.com/your-username/ROOTX-Cipher/releases) page and follow the setup wizard.
